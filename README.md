## Setup
- Add `nuxt-swagger-module` dependency using yarn or npm to your project
- Add `nuxt-swagger-module` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    'nuxt-swagger-module',

    // With options
    ['nuxt-swagger-module', { /* module options */ }],
 ]
}
```
